Simple Tab Pane

Original Page

Demo for a simple tab pane MooTools plugin. Below you can see a simple tab pane with three tabs, enabling a user to switch between three pieces of content.
Due to the nature of the inner workings of the tab pane, it will remain functional even after completely reloading it with an asynchronous request. As long as the container the pane is created with stays on the page, the tab pane will remain intact. You can simulate this on this page by adding a new tab using the form on the bottom of the page.

For more information and documentation see the MooTools Forge page for this plugin or the github page.
