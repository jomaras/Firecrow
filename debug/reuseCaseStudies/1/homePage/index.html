  • Firecrow

    Firecrow is a Firefox plugin developed on top of Firebug for extracting reusable client-side UI controls. The main idea is that the user selects the disired UI control and performs a series interactions that demonstrate the desired behavior. During the interaction, Firecrow monitors the executed code, applied CSS styles, used resources and extracts everything that is need to reuse the chosen control. »

  • Firedoc

    Firedoc is a Firefox plugin prototype developed on top of Firebug for analyzing the execution of client-side JavaScript code. By connecting to the JavaScript debugger i gathers the execution log, and displays all executed JavaScript code expressions (not just lines). »

  • phpModeler

    phpModeler is a reverse engineering tool that statically analyses web application source code (PHP scripts, HTML pages, and JavaScript libraries) and generates models that can be used as a basis for architecture recovery; it generates dependency models that show interdependencies between web application elements and visualizes web page evolution. It is realized as an Eclipse plugin. »
